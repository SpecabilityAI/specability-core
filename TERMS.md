# Preview Terms

Specability Core preview builds are provided for local evaluation and workflow
feedback.

## Local Use

Specability Core runs locally. The CLI does not automatically upload source
code, runtime state, or workflow data to SpecabilityAI.

## Feedback

Feedback is explicit and user-selected. Do not submit confidential source code,
secrets, customer data, private logs, or material you are not authorized to
share.

By submitting feedback through this repository, you grant SpecabilityAI
permission to use that feedback to improve Specability Core, related method
packs, and future Specability products.

Preview builds are provided as-is, without warranties of any kind.
